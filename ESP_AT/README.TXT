--------------------------------------------------------------------------------------------------------------
ВНИМАНИЕ! 
--------------------------------------------------------------------------------------------------------------

1. Прошивка тестировалась под версией Arduino IDE 1.8.5;
2. Архив с прошивкой должен быть распакован, чтобы ВСЕ ФАЙЛЫ лежали в папке с именем "ESP_AT" (без кавычек).

--------------------------------------------------------------------------------------------------------------
ИНСТРУКЦИЯ ПО ПРОШИВКЕ ESP
--------------------------------------------------------------------------------------------------------------

1. Открыть Arduino IDE, пойти в меню "Файл->Настройки". В поле "Дополнительные ссылки для менеджера плат"
вставить адрес http://arduino.esp8266.com/stable/package_esp8266com_index.json - затем нажать ОК,
закрыв окно;

2. Пойти в  меню "Инструменты->Плата->Менеджер плат", дождаться, пока всё прогрузится, в поле поиска ввести
строчку esp - в результатах покажется пункт "esp8266 by ESP8266 Community". Выбрать этот пункт, выбрать
нужную версию (прошивка тестировалась под версией пакета 2.4.0), нажать кнопку "Установка". Рекомендуется
после установки перезагрузить Arduino IDE;

3. Открыть в Arduino IDE файл ESP_AT.ino, выбрать нужную плату и порт, скомпилировать и закачать прошивку.

--------------------------------------------------------------------------------------------------------------
ВНИМАНИЕ! 
--------------------------------------------------------------------------------------------------------------

Скорость работы контроллера теплицы с ESP по умолчанию - 57600 бод, скорость порта менять на ту, 
что указана в настройках прошивки контроллера теплицы для работы с ESP! Для смены скорости найти в файле
прошивки ESP UART_SPEED (в самом верху) - и там выставить нужную скорость.
